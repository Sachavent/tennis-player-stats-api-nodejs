# tennis-player-stats-api-nodejs

## PREREQUIRES
You should have:
- node 10 install or nvm (nvm use or nvm install)
- Docker

## INTALLING AND RUN
- `nvm use` (to get node 10)
- `npm install`
- `npm run start`

## Tests
To run tests:
- On terminal run : `NODE_ENV=test npm run start`
- On another terminal run : `npm run test`
